PARAMETRIC AIRCRAFT GEOMETRY MODELLER

Using the CPACS file format and TiGL, you can model a wide range of aircraft, spacecraft and rotorcraft.

ENABLES ANALYSIS AND OPTIMIZATION

TiGL is used in and outside the DLR for aircraft design analysis and optimization and is used by many different analysis tools. The file exports to CAD file formats are a standardized interface to industry established software.

LANGUAGE BINDINGS

TiGL comes with bindings to popular programming languages, including Python, Matlab, and Java. You can even use TiGL together with PythonOCC to create new geometries or extend TiGL.

About

The TiGL Geometry Library (TiGL) is an open source software package for the parametric modeling of aircraft geometries. It is developed at the German Aerospace Center (DLR) at the department of High-Performace Computing. At the DLR, TiGL is used as a central tool for the analysis and optimization of aircraft, in which the aircraft geometries generated in TiGL are made available to the simulation tools via various interfaces. TiGL thus provides the link between the parametric CPACS description of the aircraft and these tools. It offers functions for exporting the geometry to common CAD formats (IGES, STEP, VTK) as well as functions for calculating points and curves on the aircraft surface.

Find an overview about TiGL on the feature page or have a look at this survey article.
